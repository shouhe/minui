# wxc-switch

> MinUI 小程序组件 - 开关选择器

## Install

``` bash
$ min install @minui/wxc-switch
```

## API

### Switch【props】

| 名称                  | 描述                         |
|----------------------|------------------------------|
|`checked`           | [Boolean] 开关状态，默认为 false（不禁用）       |
|`disabled`         | [Boolean] 是否禁用，默认 false（不禁用）  |
|`active-color`         | [String] 状态为开时的背景色，默认 #FF5777         |
|`bind:switch` | [Function] 开关切换时的回调函数 |
